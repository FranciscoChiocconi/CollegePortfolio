High School Projects

OpenCV Lane Detection

January 2020, AP Computer Science A

The objective of our project was to make find lanes on a road given a video or an image. This project was not assigned but was used for a project in this class. In this project we were using a library called OpenCV which helps with computer vision. My partner were interested on how computer vision worked so this project was perfect for the situation. To find the lanes we used many custom methods which are in the OpenCV library including hough lines, hough transform, edge detection algorithms, gaussian blur and more. Overall the project was a fun learning experience on computer vision and I would be interested in doing a similar project.
